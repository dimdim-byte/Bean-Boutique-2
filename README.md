☕ Bean Boutique
A modern, responsive coffee shop website

Bean Boutique is a stylish, multi-page web project designed to showcase a coffee shop’s offerings — including beverages, coffee equipment, events, and special promotions.
It features a clean, elegant design built with HTML, CSS, and JavaScript.

📁 Project Structure
```bash
Bean-Boutique/
├── pages/          # Contains individual HTML pages
│   ├── index.html  # Homepage
│   ├── menu.html   # Coffee menu
│   ├── equipment.html # Coffee equipment store
│   ├── cart.html   # Shopping cart
│   ├── events.html # Coffee events
│   ├── offers.html # Promotions & offers
│   └── about.html  # About the coffee shop
│
├── scripts/        # JavaScript files
│   ├── main.js     # General site logic
│   ├── script.js   # UI interactions
│   ├── menu.js     # Menu functionality
│   ├── equipment.js# Product rendering
│   └── cart.js     # Cart logic
│
├── style/          # Stylesheets
│   ├── beancoffee.css
│   └── cart.css
│
└── assets/         # Images, icons, and fonts
```
⚙️ Features

🏠 Multi-page responsive layout

☕ Interactive menu and equipment listings

🛒 Functional cart system (add/remove items)

🎟️ Event and offers pages

💬 Smooth animations and transitions

📱 Optimized for desktop and mobile devices

🧰 Built With

HTML5 — Page structure

CSS3 — Styling and layout (custom design using Google Fonts & Bootstrap Icons)

JavaScript (ES6) — DOM manipulation, events, and dynamic rendering

Bootstrap Icons — Visual icons for UI consistency

Google Fonts — “Montserrat” and “Playfair Display” for typography
